## Установка

### Требования
- Python 3.7+
- Windows (для полной функциональности)

### Зависимости
```bash
pip install -r requirements.txt
```

Основные пакеты:
- `requests` - для работы с VirusTotal API
- `tkinter` - встроенный в Python
- `wmi` (Windows) - для мониторинга USB

## Настройка VirusTotal API

### 1. Получить API ключ
1. Зайдите на https://www.virustotal.com
2. Создайте аккаунт (если его нет)
3. Перейдите в личный кабинет
4. Скопируйте API ключ

### 2. Установить переменную окружения

**Windows (PowerShell):**
```powershell
$env:VIRUSTOTAL_API_KEY="your-api-key-here"
```

**Windows (cmd):**
```cmd
set VIRUSTOTAL_API_KEY=your-api-key-here
```

**Linux/macOS:**
```bash
export VIRUSTOTAL_API_KEY="your-api-key-here"
```

### 3. Постоянная установка

**Windows:**
1. Нажмите Win+R
2. Введите `sysdm.cpl`
3. Перейдите на вкладку "Дополнительно"
4. Нажмите "Переменные среды"
5. Добавьте новую переменную с именем `VIRUSTOTAL_API_KEY`

**Linux/macOS:**
Добавьте в `~/.bashrc` или `~/.zshrc`:
```bash
export VIRUSTOTAL_API_KEY="your-api-key-here"
```

## Использование

### Быстрый запуск
```bash
python main.py
```

### Регистрация в контекстном меню Windows
```bash
python main.py register
```

Это добавит пункт "Проверить с Aegis Core" при нажатии правой кнопки на файл.

### Удаление из контекстного меню
```bash
python main.py unregister
```

## Функции

### Сканирование
- **Быстрая проверка** - сканирует TEMP и Downloads
- **Полная проверка** - полное сканирование системы
- **Проверка файла** - выбор конкретного файла
- **Проверка USB** - при подключении USB устройства

### Работа с угрозами
- 👁️ **Просмотр** - просмотр списка найденных угроз
- 📦 **В карантин** - безопасное изоляция файла
- 🏥 **Лечение** - попытка лечения файла (санитизация)
- 🗑️ **Удаление** - удаление файла (с резервной копией или безопасно)

## Ограничения

- Требуется Windows для полной функциональности USB мониторинга
- VirusTotal API имеет лимиты на количество запросов
- Некоторые защищенные системные файлы нельзя удалить без перезагрузки

## Решение проблем

### Нет звука
1. Проверьте громкость системы
2. Убедитесь, что звук включен в параметрах Windows
3. Переопределите переменные окружения

### VirusTotal не работает
1. Проверьте интернет-соединение
2. Проверьте правильность API ключа
3. Проверьте переменную окружения `VIRUSTOTAL_API_KEY`

### Не удается удалить файл
1. Закройте программу, использующую файл
2. Запустите приложение от администратора
3. Используйте опцию "Безопасное удаление"
