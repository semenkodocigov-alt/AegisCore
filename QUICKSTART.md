## 2. НАСТРОЙКА VIRUSTOTAL (ОБЯЗАТЕЛЬНО ДЛЯ ПОЛНОЙ ФУНКЦИОНАЛЬНОСТИ)

### 2.1 Регистрация
1. Зайдите на: https://www.virustotal.com/gui/home/upload
2. Нажмите "Sign up" (или создайте акаунт)
3. Подтвердите email

### 2.2 Получение API ключа
1. После регистрации, перейдите в профиль (значок пользователя)
2. Выберите "API Key"
3. Скопируйте ваш API ключ (выглядит как длинная строка)

### 2.3 Установка переменной окружения (Windows)

**Способ 1 - PowerShell (временно):**
```powershell
$env:VIRUSTOTAL_API_KEY="ваш-api-ключ-здесь"
python main.py
```

**Способ 2 - Постоянно (Системные переменные):**
1. Нажмите Win+R
2. Введите: `sysdm.cpl`
3. Перейдите на вкладку "Дополнительно"
4. Нажмите "Переменные среды"
5. Нажмите "Создать" (для переменных пользователя)
6. Имя: `VIRUSTOTAL_API_KEY`
7. Значение: (вставьте ваш API ключ)
8. Нажмите OK
9. Перезагрузитесь

**Способ 3 - Через cmd:**
```cmd
setx VIRUSTOTAL_API_KEY "ваш-api-ключ-здесь"
```
(Потребуется перезагрузка)

### 2.4 Проверка
```powershell
# PowerShell
$env:VIRUSTOTAL_API_KEY

# cmd
echo %VIRUSTOTAL_API_KEY%
```

Если видите ваш ключ - всё работает!

## 4. ИСПОЛЬЗОВАНИЕ

### 4.1 Запуск приложения
```bash
python main.py
```

### 4.2 Основные функции

#### Сканирование
- **⚡ Быстрая проверка** - сканирует TEMP и Downloads (~30 сек)
- **🔍 Полная проверка** - полное сканирование системы (~5-10 мин)
- **📄 Проверка файла** - проверить конкретный файл

#### Работа с угрозами
1. **Просмотр** - кнопка "Просмотр" показывает все найденные файлы
2. **В карантин** (📦) - безопасно изолировать файл
3. **Лечение** (🏥) - попытка удалить вредоносный код
4. **Удаление** (🗑️) - удалить файл (с резервной копией!)
5. **Проверка подписи** - проверить подлинность от Microsoft

#### Дополнительно
- **🔌 USB Защита** - автоматически сканирует подключаемые USB
- **🗂️ Очистка кэша** - удалить кэш приложений
- **📦 Карантин** - просмотр и управление изолированными файлами

### 4.3 Примеры использования

**Сценарий 1 - Быстрая проверка системы:**
1. Запустите приложение
2. Нажмите "⚡ Быстрая проверка"
3. Ждите результатов
4. Если найдены угрозы - выберите файл и нажмите "🗑️ Удалить"

**Сценарий 2 - Проверка скачанного файла:**
1. Нажмите "📄 Проверить файл"
2. Выберите файл
3. Дождитесь результата
4. При необходимости удалите или поместите в карантин

**Сценарий 3 - Включение USB мониторинга:**
1. Нажмите "🔌 USB Защита" (кнопка станет зеленой "ВЫКЛ")
2. Подключите USB устройство
3. Приложение автоматически его просканирует

## 5. ВОЗМОЖНЫЕ ПРОБЛЕМЫ

### "VirusTotal не работает"
❌ Ошибка: API ключ не найден
✅ Решение:
1. Проверьте переменную окружения (см. п. 2.4)
2. Перезагрузитесь
3. Проверьте правильность ключа

❌ Ошибка: Rate limit exceeded
✅ Решение: Подождите несколько минут (лимит 4 запроса/минуту для бесплатного API)

### "Нет звука"
❌ Ошибка: Звук не воспроизводится
✅ Решение:
1. Проверьте громкость Windows
2. Проверьте что звук включен в "Параметры -> Звук"
3. Перезагрузитесь

### "Не удается удалить файл"
❌ Ошибка: "Permission denied"
✅ Решение:
1. Закройте программу, которая использует файл
2. Запустите приложение от администратора
3. Используйте "Безопасное удаление"

### "WMI недоступен" (для USB мониторинга)
❌ Ошибка: WMI не работает
✅ Решение:
1. Установите: `pip install pywin32`
2. Запустите: `python -m pywin32_postinstall -install`

## 7. ПОНИМАНИЕ РЕЗУЛЬТАТОВ СКАНИРОВАНИЯ

### Типы угроз

**🔴 CRITICAL (Критично)**
- Ransomware (шифровальщики)
- Trojan (трояны)
- Worm (черви)
- Требуется немедленное удаление

**🟠 HIGH (Высокий)**
- PUP (нежелательное ПО)
- Adware (рекламное ПО)
- Suspicious files (подозрительные файлы)
- Рекомендуется удалить

**🟡 MEDIUM (Средний)**
- Potentially unwanted programs
- Suspicious behavior
- Requires further investigation

**🟢 LOW (Низкий)**
- Suspicious names
- Harmless detections
- Monitor

## 9. БЕЗОПАСНОСТЬ

### Где хранятся данные?
- **Резервные копии**: `./backups/` (у вас на диске)
- **Кэш VirusTotal**: `./cache/signatures_cache.json` (локально)
- **Логи**: В приложении (опционально в файл)

### Конфиденциальность
- Хеши файлов отправляются на VirusTotal (стандартная практика)
- Оригинальные файлы НЕ отправляются
- Данные сохраняются локально
