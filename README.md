# Aegis Core

Aegis Core — это простое антивирусное приложение на Python с графическим интерфейсом на Tkinter. Проект умеет проверять файлы и директории на основе сигнатур и простого эвристического анализа.

## Особенности

- Графический интерфейс на `tkinter`
- Быстрая и полная проверка
- Проверка отдельного файла через диалог выбора
- Логирование работы и вывод найденных угроз
- Список обнаруженных заражённых файлов в интерфейсе
- Карантин для перемещения подозрительных объектов
- Поддержка сборки в один исполняемый файл через `PyInstaller`

## Установка

1. Убедитесь, что установлен Python 3.10+.
2. Установите зависимости, если они есть в проекте:

```bash
pip install -r requirements.txt
```

Если `requirements.txt` отсутствует, установите только стандартные модули: `tkinter` (обычно входит в дистрибутив Python).

## Запуск

Из корневой папки проекта запустите:

```bash
python main.py
```

## Использование

- Нажмите `Быстрая проверка` для сканирования временных папок и загрузок
- Нажмите `Полная проверка` для глубокой проверки системных и пользовательских директорий
- Нажмите `Проверить файл`, чтобы выбрать файл вручную
- В боковой панели отображаются найденные угрозы и лог выполнения

## Структура проекта

- `main.py` — точка входа приложения
- `gui_app.py` — основное графическое приложение Tkinter
- `scanner.py` — логика сканирования файлов и директорий
- `signatures.py` — работа с базой сигнатур
- `analyzer.py` — эвристический анализ файлов
- `quarantine.py` — управление карантином
- `hash_utils.py` — вычисление хешей файлов
- `core/` — модульное ядро антивирусного движка
- `gui/` — альтернативная GUI-реализация и дополнительные окна

## Сборка в exe

Если хотите собрать приложение в один исполняемый файл на Windows, используйте `PyInstaller`:

```bash
pyinstaller --noconsole --onefile main.py
```

## Примечания

- Проект работает на Windows и ориентирован на локальное тестирование.
- `tkinter` должен быть доступен в вашей установке Python.
- Для более сложного антивирусного движка рекомендуется добавить дополнительные сигнатуры и улучшить эвристику.
